#  Лабораторная работа №1: Системы обмена данными и основы Git

##  Описание
Данная лабораторная работа посвящена изучению распределённых систем контроля версий (Git), настройке рабочего окружения разработчика и созданию CLI-приложений с помощью библиотеки `typer`. В рамках задания выполняются:
- Настройка GitHub и интеграция с `gh CLI`
- Генерация и привязка ключей (SSH, PAT, GPG)
- Работа с ветками, Pull Request, `rebase` и подписью коммитов
- Разработка CLI-утилиты с аргументами, опциями и форматированным выводом
