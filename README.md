#  DevSecOps Course Labs

Курс по прикладной безопасности приложений и DevSecOps-практикам.  
В ходе выполнения лабораторных работ вы освоите инструменты для безопасной разработки, анализа уязвимостей и автоматизации проверок в CI/CD.

---

##  Список лабораторных работ

| № | Лабораторная работа | Тема |
|---|---------------------|------|
| 01 | `lab1` | GitSCM — подготовка рабочего окружения |
| 02 | `lab2` | *nix — права доступа, SUID, ACL, процессы |
| 03 | `lab3` | Nmap — сканирование сети, NSE и защита результатов |
| 04 | `lab4` | Анализ и определение мер снижения рисков ИБ |
| 05 | `lab5` | Docker — контейнеризация приложений |
| 06 | `lab6` | Docker CIS Benchmark и Trivy |
| 07 | `lab7` | SAST, SCA и Secret Detection |
| 08 | `lab8` | DAST — OWASP ZAP и ручное тестирование |
| 09 | `lab9` | DevSecOps CI/CD конвейер на GitHub Actions |
| 10 | `lab10` | Оценка анализа рисков ИБ — практика |
